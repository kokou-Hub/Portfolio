Emergency Room Visits Analysis

This Emergency Room Visits Analysis project explores patient visit trends, wait times, and satisfaction scores in a healthcare setting. Using Python for data preprocessing, Power BI for visualization, Power Query for data transformation, and Excel for initial structuring, this project transforms raw emergency room data into actionable insights, helping to identify busiest hospital hours, department efficiency, and patient satisfaction levels. The interactive dashboard enables real-time exploration of key metrics, allowing healthcare professionals to make data-driven decisions to improve patient experience and hospital operations.
