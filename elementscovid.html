Resources

1. Dataset Information

  • Description of the Dataset: The dataset provides insights into food consumption trends before, during, and after the COVID-19 pandemic, highlighting category-level consumption patterns.
  • Data Source: The dataset was sourced from Kaggle and analyzed to track changes in consumer behavior over different pandemic phases.
  • How the Data Was Used:
    • Data Cleaning & Preparation: Processed in Excel to remove inconsistencies and standardize formats.
    • Data Transformation: Aggregation and structuring before importing into Tableau for visualization.
    • Data Visualization: Interactive Tableau dashboards created to analyze key insights.

2. Tools and Technologies

  • Excel: Used for initial data structuring and preparation.
  • Tableau: Used for interactive visualization and dashboard creation.
