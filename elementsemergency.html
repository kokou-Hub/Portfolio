Resources

1. Dataset Information

  • Description of the Dataset: The dataset provides insights into emergency room visits, focusing on patient demographics, visit trends, satisfaction scores, and wait times across various departments.
  • Data Source: The dataset was sourced from Kaggle, ensuring anonymity while preserving valuable insights into emergency room visits.
  • How the Data Was Used:
    • Data Cleaning & Preparation: Handled in Excel and Power Query to remove inconsistencies.
    • Data Transformation: Aggregated and structured for visualization in Power BI.
    • Data Visualization: Developed in Power BI to present key trends and insights interactively.

2. Tools and Technologies

  • List of Tools Used:
    • Power BI: For creating interactive dashboards and visualizations.
    • Excel: Used for initial data cleaning and organization.
    • Power Query: Performed data transformation and preprocessing.
    • PowerPoint: Designed visually appealing dashboard templates for Power BI integration.
